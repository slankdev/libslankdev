
# libnl sample codes

install libnl
```
$ sudo apt update && sudo apt install \
    pkg-config libnl-3-200 libnl-3-dev \
    libnl-cli-3-200 libnl-cli-3-dev \
    libnl-genl-3-200 libnl-genl-3-dev \
    libnl-route-3-200 libnl-route-3-dev \
    libnl-nf-3-200 libnl-nf-3-dev
```

uninstall libnl
```
$ sudo apt remove \
    pkg-config libnl-3-200 libnl-3-dev \
    libnl-cli-3-200 libnl-cli-3-dev \
    libnl-genl-3-200 libnl-genl-3-dev \
    libnl-route-3-200 libnl-route-3-dev \
    libnl-nf-3-200 libnl-nf-3-dev
```
